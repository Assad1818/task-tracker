# Task Tracker - Java

Aplicação em Java para gerenciamento de tarefas via terminal.

## Funcionalidades
- Adicionar tarefas
- Atualizar status (Para fazer, Em progresso, Concluída)
- Remover tarefas
- Listar tarefas por status

## Tecnologias
- Java
- Programação Orientada a Objetos
- Enum
- Collections

## Como executar
1. Compile os arquivos Java
2. Execute a classe `TaskTracker`
